### 2. AI 功能（需要后端支持）

**重要提示**: AI对话功能需要后端服务支持。

**原因**: 
- 前端代码调用 `http://localhost:8080/api/ai/stream` API
- DashScope API 有 CORS 限制，无法从前端直接调用
- 必须通过后端服务代理 AI API 请求

**解决方案**:
1. **启动后端服务** (推荐)
   - 后端服务需要运行在 `http://localhost:8080`
   - 后端提供 `/api/ai/stream` 接口
   - 后端负责调用 DashScope API 并返回流式响应

2. **最小化后端服务**
   - 只需要启动包含 AI Controller 的部分
   - 不需要连接数据库和 Redis
   - 只需要 DashScope API 密钥配置

## 🔍 故障排查

### 前端服务无法启动
1. 检查端口 5173 是否被占用
2. 检查 Node.js 版本是否正确
3. 清除 node_modules 后重新安装依赖

### AI 功能无法使用
1. 检查后端服务是否启动
2. 检查后端是否运行在 `http://localhost:8080`
3. 检查浏览器控制台错误信息
4. 检查 DashScope API 密钥配置

### 页面无法访问
1. 检查前端服务是否正常运行
2. 检查浏览器控制台错误
3. 检查路由配置
